# QuizMaster - Interactive Quiz Web App

A complete quiz application built with vanilla JavaScript, HTML, and Tailwind CSS.

## Features

- **User Authentication**: Sign up and sign in with validation
- **Profile Management**: View user stats and quiz history
- **Quiz Selection**: Browse quizzes by category (HTML, CSS, JavaScript, React, Node.js)
- **Interactive Quizzes**: Timed questions with progress tracking
- **Results Review**: Detailed feedback on incorrect answers
- **Local Storage**: Persistent data storage

## Pages

1. **Landing Page** - Welcome page with app introduction
2. **Authentication** - Sign up/Sign in with validation
3. **Profile** - User dashboard with statistics
4. **Quiz Selection** - Browse and filter available quizzes
5. **Quiz Taking** - Interactive quiz with timer and navigation
6. **Results Review** - Detailed results and incorrect answer review

## Quiz Data

- 5 different quizzes (HTML, CSS, JavaScript, React, Node.js)
- 10 questions per quiz
- Multiple choice format
- Category-based filtering

## Technical Implementation

- **Frontend**: Vanilla JavaScript, HTML5, Tailwind CSS
- **Storage**: localStorage for user data and quiz results
- **Architecture**: Modular JavaScript with separate files for each feature
- **Responsive**: Mobile-friendly design

## Getting Started

1. Open `src/temp.html` in a web browser
2. Sign up for a new account or use existing credentials
3. Browse available quizzes
4. Take quizzes and review your results
5. Track your progress in the profile section
